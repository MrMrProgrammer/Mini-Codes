```markdown
clear(): پاک کردن محتوای یک المان، مانند یک ورودی متنی.
click(): کلیک روی یک المان، مانند یک دکمه.
find_element(by, value): یافتن یک المان بر اساس استراتژی و مقدار دلخواه.
find_element_by_class_name(name): یافتن یک المان بر اساس نام کلاس آن.
find_element_by_css_selector(selector): یافتن یک المان بر اساس یک سلکتور CSS.
find_element_by_id(id): یافتن یک المان بر اساس شناسه آن.
find_element_by_link_text(text): یافتن یک لینک بر اساس متن آن.
find_element_by_name(name): یافتن یک المان بر اساس نام آن.
find_element_by_partial_link_text(text): یافتن یک لینک بر اساس بخشی از متن آن.
find_element_by_tag_name(name): یافتن یک المان بر اساس نام تگ آن.
find_element_by_xpath(xpath): یافتن یک المان بر اساس مسیر XPath.
find_elements(by, value): یافتن همه المان‌های مطابق با استراتژی و مقدار داده شده.
find_elements_by_class_name(name): یافتن همه المان‌هایی که کلاس آن‌ها با نام داده شده مطابقت دارد.
find_elements_by_css_selector(selector): یافتن همه المان‌هایی که با یک سلکتور CSS مطابقت دارند.
find_elements_by_id(id): یافتن همه المان‌هایی که شناسه آن‌ها با شناسه داده شده مطابقت دارد.
find_elements_by_link_text(text): یافتن همه لینک‌هایی که متن آن‌ها با متن داده شده مطابقت دارد.
find_elements_by_name(name): یافتن همه المان‌هایی که نام آن‌ها با نام داده شده مطابقت دارد.
find_elements_by_partial_link_text(text): یافتن همه لینک‌هایی که بخشی از متن آن‌ها با متن داده شده مطابقت دارد.
find_elements_by_tag_name(name): یافتن همه المان‌هایی که تگ آن‌ها با نام داده شده مطابقت دارد.
find_elements_by_xpath(xpath): یافتن همه المان‌هایی که مسیر XPath آن‌ها با مسیر داده شده مطابقت دارد.
get(url): باز کردن یک صفحه وب با آدرس داده شده.
get_attribute(name): دریافت مقدار یک ویژگی مشخص از یک المان، مانند value یک ورودی متنی یا href یک لینک.
get_cookies(): دریافت کوکی‌های فعلی.
get_screenshot_as_file(filename): گرفتن عکس از صفحه وب و ذخیره آن به عنوان یک فایل.
get_screenshot_as_png(): گرفتن عکس از صفحه وب و بازگرداندن آن به صورت یک رشته بایت PNG.
is_displayed(): بررسی مشاهده شدن یک المان.
is_enabled(): بررسی فعال بودن یک المان.
is_selected(): بررسی انتخاب شدن یک المان.
quit(): بستن مرورگر.
refresh(): بازسازی صفحه فعلی.
save_screenshot(filename): گرفتن عکس از صفحه وب و ذخیره آن به عنوان یک فایل.
send_keys(*keys_to_send): ارسال کلیدهای کیبورد به المان مشخص شده، مانند یک ورودی متنی.
set_window_position(x, y): تنظیم موقعیت پنجره مرورگر.
set_window_size(width, height): تنظیم اندازه پنجره مرورگر.
switch_to.active_element: انتقال کنترل به المان فعال.
switch_to.alert: تعامل با پنجره هشدار.
switch_to.default_content(): بازگشت به فریم پیش‌فرض.
switch_to.frame(frame_reference): تعیین فریم مورد نظر برای اجرای عملیات.
switch_to.parent_frame(): بازگشت به فریم والد.
switch_to.window(window_name): تغییر به پنجره با عنوان داده شده.
```
